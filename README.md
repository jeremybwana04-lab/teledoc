# Teledoc

A website that connects patients to licensed doctors and medical professionals through video calls and other methods to improve healthcare access, especially in Kenya. The service is free to users.

## Features
- Video consultations with licensed medical professionals
- Secure messaging and patient-doctor communication
- Appointment scheduling and reminders
- Basic patient profile and medical history
